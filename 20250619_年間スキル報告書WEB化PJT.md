# memory-bank

### memory-bankとは
**AIエージェントがプロジェクトの文脈（コンテキスト）や設計・進捗情報を長期的かつ構造的に保持・参照するためのドキュメント群**

### 構成ファイル
- **projectbrief.md**：プロジェクトの基本情報、目的、要件
- **productContext.md**：解決する課題、ユーザー体験、業務フロー
- **systemPatterns.md**：システムアーキテクチャ、設計パターン
- **techContext.md**：使用技術、開発環境、技術的制約
- **activeContext.md**：現在の開発フォーカス、最近の変更
- **progress.md**：実装済み機能、未実装機能、進捗状況
